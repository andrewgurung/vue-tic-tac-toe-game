
# Vue Tic-Tac-Toe Game
Building a Tic-Tac-Toe Game with Vue 2

Author Info
-----------
Author: Andrew Gurung <br>

Build Setup
-----------

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
